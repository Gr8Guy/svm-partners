Professional Partners Dashboard

This dashboard is created to be a tool for our professional partners, with simplified menus and direct links to resources and contacts within the school. If you need the full menu of options, please visit the TTU SVM Mainpage.

This website has been optimized to work on your mobile device at your convenience, by following the instructions below you will be able keep this website bookmarked and installed on your homescreen just like a native application.

How to install a shortcut to this website on your mobile device.

For iPhone & iPad

  1. While viewing the website on Safari, tap in the menu bar.
  2. Scroll down the list of options, then tap Add to Home Screen.
  3. On the next prompt that appears, tap Add to confirm
  4. If you don't see Add to Home Screen, you can add it. Scroll down to the bottom of the list, tap Edit Actions, then tap Add to Home Screen

For Android

  1. Open Chrome.
  2. Navigate to the website or web page you want to pin to your home screen.
  3. Tap the menu icon (3 dots in upper right-hand corner) and tap Add to homescreen.
  4. Choose a name for the website shortcut, then Chrome will add it to your home screen.
